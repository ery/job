[杭州] 淘宝招聘 Ruby开发工程师

招聘 • huacnlee • 于发布 • 最后由 Vizwind 于回复 • 4861次阅读

我也发个招聘吧，要人的是目前我所在的部门，一淘 UED 团队。
现在还要 2,3 个人，工作地点：杭州
淘宝的待遇不错的！此外我们部门 Ruby 工程师是全 Mac，可以选择 Macbook Pro 或是 iMac。

要求

  1. 清楚自己为什么使用 Ruby，体会到 Ruby 编程的乐趣，尝试过各种 Ruby-Style 的编程，疯狂的 Ruby 爱好者；
  2. 有丰富的 Linux 使用经验，必要时能够承担起 APP OPS 的职责
  3. 读过 Rails 源码，了解 Rails 的基本代码结构，对历史版本的 Rails 有足够的了解。贡献过补丁或者代码者尤佳，写过 Gem 者尤佳；
  4. 对可伸缩 Web 的部署和开发有心得，喜欢尝试新架构；
  5. 熟悉 MySQL，熟悉 Memcached，有 NoSQL 使用经验者尤佳！
  6. 你要是全能型选手

简历直接投 @nowazhu 邮箱: nowazhu@gmail.com (PS: 用 PDF 格式发送，并带上你的博客，Twitter, Github 地址)
