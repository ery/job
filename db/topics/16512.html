[远程] 招聘 Ruby on Rails 工程师

招聘 • wing328 • 于发布 • 最后由 tyaccp_guojian 于回复 • 901次阅读

职责：
- 负责系统的架构设计、研发工作
- 设计软件测试用例以提升软件质量
- 探索新技术以提高服务水平

必需具备的技能:
- 3年以上Ruby on rails开发经验 (经验较少者将被招聘为初级工程师)
- 熟悉 Linux, git (我们使用 Ubuntu, BitBucket)
- 对 BDD, TDD 有基本的了解
- 对 HTML, CSS, Javascript 有基本的了解 (我们使用 Twitter Bootstrap)
- 善于学习，良好的沟通技巧
- 独立工作能力

加分:
- 有 Redis、ElasticSearch 经验
- 有 Postgresql 经验
- 有 NoSQL (e.g. CouchDB) 经验
- Web API (Twitter, Facebok, Twilo, Flickr) 开发经验
- 有服务导向架构(Service-Oriented Architecture,SOA) 经验

我们是一家香港创业公司。创业团队拥有6年以上的顶级投资银行工作经验。目前我们正在开发一种平台解决方案，以简化WebAPI发展。所有团队成员都是远程工作：http://37signals.com/remote/

开发理念: http://theleanstartup.com/principles

下面是一些我们使用的技术：

Ruby on Rails (v4)
Ember.js
Redis
PostgreSQL
ElasticSearch
DigitalOcean (cloud hosting platform)

请将您的简历和期望薪资电邮给 wing328(at)yahoo.com

As a Ruby on Rails engineer, you will be actively involved in the development of our platform solution to streamline Web API development.

Duties:
- contribute to the architecture and evolution of the platform
- design and implement new features based on requirement
- devise test cases to enhance software quality
- explore new technologies to improve the service level

Requirements:
- 3 years of experience with Ruby on Rails. (candidate with less experience would be considered as a junior engineer)
- knowledge with BDD or TDD (rspec, factory_girl)
- knowledge with CSS, HTML, Javascript (we use Twitter Bootstrap)
- familiar with Linux, git (we use Ubuntu, bitbucket)
- fast learner with good communication skill
- ability to work independently

Bonus:
- experience with Redis, ElasticSearch
- experience with Postgresql and scaling database (e.g. sharding)
- integration with third-party Web API (Twitter, Facebook, Twilo, Flickr) is a plus
- experience with service-oriented architecture
- experience with NoSQL (e.g. CouchDB)

Company background:
We are a startup based in Hong Kong. The founding team has 6+ years of work experience with top tier investment bank. Currently we are working on a platform solution to streamline WebAPI development. All team members work remotely: http://37signals.com/remote/. Our development principle is based on Lean Startup Methodology: http://theleanstartup.com/principles.

Here are some of the technologies we use:

Ruby on Rails (v4)
Ember.js
Redis
PostgreSQL
ElasticSearch
DigitalOcean (cloud hosting platform)

Please submit your resume and expected salary (in RMB) to wing328(at)yahoo.com.
