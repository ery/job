[北京] 天际网寻觅初、中、高级 Rubyist [暂停招聘]

招聘 • chechaoyang • 于发布 • 最后由 chechaoyang 于回复 • 4125次阅读

目前天际网 Ruby 团队已达到 18 人，公司已暂停招聘，待下一步有招聘需求时再继续进行。

距离我们上一次招聘已经过去好几个月了，随着公司的发展，现在我们又需要新鲜血液了！

我们是谁？
我们是由 18 个 Rubyist 组成的团队，行业背景五花八门，有威武的大牛，也有粉嫩的新人。我们的共同点是：都热爱新技术，喜欢尝试新事物。就像每一个加入到 Ruby 社区的同学一样，我们有开放的心态，希望自己能利用这大好的人生做点什么！我们深有同感的是：Life is short, use Ruby.（借用自邻居 Python 社区）

我们在做什么？
天际网（ http://www.tianji.com ）是中国最大的职业社交网站，其主要用户群体为职业人士，满足用户的商业和职业方面的需要，帮助用户更有效地建立、管理、拓展人际关系网，从而积累职业生涯中最具价值的财富。截止到2012年9月，天际网拥有注册用户1200万，每月新增25万职业人士注册体验由天际网带给他们的职业价值。

天际网发展迅速，在2009年与全球两大职业SNS之一的Viadeo合并，成为Viadeo集团全球战略的的重要组成部分。欧洲Viadeo成立于2004年，目前全球拥有3500万会员，Viadeo集团是除LinkedIn以外，唯一的一家全球性的职业SNS网站，其战略定位是欧洲+以中国为核心的成长中国家。

我们在使用
* Ruby 1.9.3
* Rails 3.2.8
* MySQL Cluster（2 sql + 2 data）
* MongoDB Cluster（2 mongos + 2 mongod）
* Redis Replication
* GlusterFS Replication
* Resque & Scheduler（4 + 2 node）
* Capistrano、Rspec......

我们在找谁？
如果你或者你身边有朋友在找 Ruby 开发的工作，请千万来我们这里看看。

福利待遇

* 前台MM、行政MM、HR MM、UI MM、前端MM、产品MM、运营MM、市场MM、销售MM（强烈建议非单身人士忽略此条！）

招聘人数
0人

薪水
这次我们招聘，为初级工程师提供的薪水是7k - 8k，中级和高级的薪水上浮的空间挺大的。技术水平上，从小工到专家，中间有很多个阶段，不管你处于哪个阶段，我们给你的回报一定会对得起你的付出的。

简历提示

3. 邮件正文以外，请发送一份 PDF 或者 Word 格式的简历

B2C 高级产品经理

岗位描述：

1、负责产品的策划与创新，能独立制定产品计划,整体把控整个产品。组织管理产品小组，协助开发团队理解和掌握需求，对产品的推进，执行的效果等KPI负责。
2、独立完成从产品规划、监督设计，协调开发到正式上线的项目管理全过程；持续跟踪产品线的数据，分析产品达标情况，制定产品改进计划。
3、负责产品线之间的协调，与其他产品团队沟通,确保网站整体的产品的推进能够同步协作进行。
4、负责跨部门协调和沟通，推动UI、开发、运营等人员紧密合作如期顺利达成产品目标。
5、收集产品需求，对用户需求、市场需求和业务需求进行调研分析，不断优化产品，提升产品质量，提高用户活跃度。

要求如下：

1、本科以上学历，四年以上工作经验,三年以上互联网产品设计、规划、项目管理等相关经验。
2、对互联网产品有深刻理解，并长期关注和研究。
3、出色的产品规划能力，优秀的口头表达能力，良好的沟通协调能力和责任心。
4、工作积极主动，执行力强,能带领团队并让团队在工作中共同提高。
5、有独立负责SNS或社区产品相关经验优先。
6、有创新精神，头脑活跃。具有良好的职业道德和职业操守及良好的团队协调能力。
7、优秀的学习能力,勇于挑战,迅速学习并掌握新的工作所需知识,迅速把控未接触过的新的类型的产品的能力 。
