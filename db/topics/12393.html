[上海] 南京西路商业圈寻找 Ruby 程序员, 待遇优厚，福利多！附环境照~

招聘 • kingboo • 于发布 • 最后由 shawzt 于回复 • 4220次阅读

职位描述：

Ruby程序员主要工作是开发和维护我们的网站和APP的服务器端。
根据应聘者能力的不同负责不同难度的工作。
我们的后端主要采用以下技术：Ruby on Rails，Mysql，Mongodb，Redis，Unicorn，Nginx。

待遇和福利：

  1. 根据实际工作能力给予20k及上不封顶的月薪。
  2. 公司配备至少双显示器等硬件设备。
  3. 免费的零食、饮料和按摩服务。
  4. 做五休二，并有免费晚餐提供。
  5. 公司的各种娱乐项目。

任职要求：

  1. 热爱编程，热爱生活。
  2. 熟悉Ruby on Rails，并有自己的Ruby on Rails项目。 【能够在线访问的请提供网址，不能的话请将源代码打包为简历附件】
  3. 熟悉Ubuntu和Git，并熟悉Git的代码提交流程。
  4. 熟悉Mysql、Mongodb、Redis中至少一种数据库，并对另外两种有所了解。
  5. 具备良好的代码书写规范，善于思考，能独立分析和解决问题。

加分：

  1. 有Github账户并参与过开源Ruby项目
  2. 熟悉sass、slim、haml、coffeescript等前端技术
  3. 同时掌握php或java等另外一门后端语言
  4. 了解用户体验，可以提出和实施UE方面的改进
  5. 了解SEO以及其他网络营销知识
  6. 了解CDN等网络加速技术

觉得自己可以胜任的 ，自信的投起你的简历吧，记得投递职位+姓名至：kingboo@neonan.com
