[杭州] 阿里巴巴继续招两个 Ruby 工程师

招聘 • huacnlee • 于发布 • 最后由 huacnlee 于回复 • 2625次阅读

2013-11-13 更新，继续招聘两名

  • 公司：阿里巴巴
  • 部门：Mac Labs
  • 地点：杭州

要求：

  • 3 年以上 Ruby, Rails 开发经验；
  • 熟悉前端的系列东西，能独立完成前端的开发工作；
  • 熟悉 Mac, Git；
  • 熟悉 MySQL, MongoDB, Redis 等数据库；
  • 熟悉 Linux 服务器的安装与维护；
  • 熟悉 Rails 项目部署；
  • 熟悉 iOS, Mac 开发优先，这边有机会参与这类项目；

工作内容

  • 负责某些创新应用的 Web 端，API 端等开发工作；
  • 维护目前团队的一些内部服务，例如 Mac, iOS App 升级包分发；

待遇

  • 基本工资得根据面试评定后才能确定
  • 年终奖至少 3 个月的薪资，偶尔公司情况好还有一些大红包
  • 外地的每年有来回机票报销
  • 各种培训机会的报销
  • 有 Mac 用
  • 面试评定过后，视情况而定，可能会有一定的期权（这个现在行情价格一股 $17.5）
  • 什么社保公积金什么的一样不少，反而交得太多会让人心疼的
  • 上班不打卡的
