[上海] CloudFoundary 部门招Web方面美工和设计师

招聘 • iBachue • 于发布 • 最后由 Junjie 于回复 • 992次阅读

我们公司 VMware Shanghai R&D Center（地址: 上海五角场创智天地） 的 CloudFoundry(http://www.cloudfoundry.com/)部门招收美工和设计师，要求如下：

Visual Designer (UI + UX)
• Tools: Adobe Photoshop + Illustrator, and wireframe tools like OmniGraffle
• Technologies: HTML/CSS
• Excellent communication skills in Chinese (and competent in English)
• Portfolio of previous work
• Top 5 list of your favorite website designs

Frontend Engineer (UI + UX)

• Tools: Chrome Web Inspector, Firebug
• Technologies: HTML/CSS, Javascript (jQuery + MVC libraries like backbone.js)
• Excellent communication skills in Chinese (and competent in English)
• Portfolio of previous work
• Top 5 list of your favorite website designs

有意者请发送Email到 zhour # vmware.com 与我联系 谢谢

下面是Cloudfoundry的介绍：

Cloud Foundry是一个开源的平台即服务，它提供给开发者自由度去选择云平台，开发框架和应用服务。Cloud Foundry最初由 VMware 发起，得到了业界广泛的支持，它使得开发者能够更快更容易的开发，测试，部署和扩展应用。Cloud Foundry是一个 开源项目 ，用户可以使用多种私有云发行版，也可以使用公共云服务，包括 CloudFoundry.com。
更多信息请见：http://www.cloudfoundry.cn/about

下面是VMware的介绍：

VMware（纽约证交所代码：VMW）在虚拟化和云计算基础架构领域处于全球领先地位，所提供的经客户验证的解决方案可通过降低复杂性以及更灵活、敏捷地交付服务来提高 IT 效率。 VMware 使企业可以采用能够解决其独有业务难题的云计算模式。 VMware 提供的方法可在保留现有投资并提高安全性和控制力的同时，加快向云计算的过度。 VMware 拥有 480,000 多家客户和 55,000 多家合作伙伴，它的解决方案可帮助各种规模的组织降低成本、提高业务灵活性并确保选择自由。
更多信息请见：http://www.vmware.com/cn/company/
