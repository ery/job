旧贴新招聘 招1名 rails工程师

职位介绍

凯业必达(CareerBuilder)是北美最大的在线网络招聘运营商，也是全球流量最大 的30个网站之一。我们正在招聘Ruby on Rails 软件工程师，前段开发工程师。我们鼓励员工的自主创新精神，帮助员工不但在工作上有机会发挥自己的创造性，同时也为员工提供良好的学习与工作的环境。凯业必达被Glassdoor.com评为2012年芝加哥50大最佳雇主之一，这也是凯业必达连续第四年获此殊荣。在50家公司中，凯业必达排名第6位，福利除了正常的社保和公积金外，我们还提供健身福利补助，额外家庭人寿保险，领导培训机会和出国出差机会。

以下是我们正在用到的几种技术：

  1. Ruby on Rails, Heroku
  2. C#, VB.NET, ASP.NET
  3. SQL, Postgres
  4. Git, TFS
  5. Web Services, CSS 3.0, JQuery, JSON
  6. 任何技术只要能让我们为Business提出更好的解决方案.

前端工程师职位需求：

  • 熟练编写结构良好的语义化的(X)HTML和浏览器兼容的CSS
  • 熟练使用javascript，编写高性能的javascript代码，了解ajax技术原理
  • 熟悉页面性能的优化，能解决各种浏览器的兼容问题
  • 有后端技术基础(.net,java,php等)，并有项目经验
  • 能够深入理解后端架构，与后端工程师配合，为项目提供最优化的技术解决方案
  • 了解SEO、用户体验；逻辑分析能力强，有良好的团队合作精神
  • 了解Ruby on Rails框架，能够编写View视图。有项目经验者优先考虑。

Rails 软件工程师职位需求：

  • 熟悉 Ruby 语言和 Ruby on Rails 框架
  • 熟悉面向对象编程，逻辑能力强
  • 主动学习并且热爱新技术
  • 有应用编程语言分析和解决问题的能力
  • 善于沟通，具备良好的团队合作精神
  • 能够按照商业需求提出好的解决方案
  • 英语熟练，CET-4及以上
