Job Description for Cloud foundry service engineer

We are seeking an experienced developer, who will build enterprise version of Cloud Foundry base on open source code (www.cloudfoundry.com). The ideal candidate is familiar with distributed system and has a strong skill of coding and debugging. Experience with developing and maintaining cloud service management system is a plus.

Requirements

  • Mastery of an OO program language like Java or C++, Ruby experience is plus.
  • Prior experience building large scale distributed system.
  • Proven troubleshooting and performance tuning experience.
  • Familiar with Linux, database, networking and scripting.
  • Familiar with agile software development practice and TDD.
  • Excellent communication skills in Chinese and English, 5+ years of experience

Job Description for Sr Tools / Release / Performance Engineer

Responsibilities: 

  • Implement, maintain, and expand SaaS tools used internally for large-scale deployments across multiple datacenters.
  • Work with engineers around the world to gather requirements and deliver scalable solutions.
  • Work closely with infrastructure teams to ensure solution robustness.

Requirements:

  • 5+ years of experience building multi-tier, scalable, enterprise products
  • Solid experience with scripting languages such as Ruby, Python, Perl, or JavaScript.
  • Solid experience with C/C++ or Java
  • Solid experience with Linux or UNIX
  • Proven ability to root cause and troubleshoot issues, identifying and delivering fixes
  • Experienced gathering high level requirements and able to create scalable solutions under realistic schedules
  • Experienced with large scale production environments and performance tuning
  • Experience with Git or Perforce a plus
  • Experience with Agile and Test Driven Development (TDD) a plus
  • Experience developing web applications is a plus.
  • Experience with VMware or other virtualization is a plus
  • Excellent written and oral communication
  • Excellent analytical and problem solving skills
  • BS or MS degree in Computer Science or equivalent
