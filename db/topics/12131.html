[杭州] 找两名初、中级 Rails 工程师

招聘 • huacnlee • 于发布 • 最后由 huacnlee 于回复 • 2694次阅读

此职位不是淘宝的，是朋友公司的。
要求不是很高，希望转行做 Ruby 的同学注意了，这是一个很好的机会！

公司情况

  • 这是一家刚起步的公司，由宁波的一家杂志社投资创办，办公地点在杭州。
  • 项目网址： http://www.yesicity.com

必要要求

  • 一年以上 Ruby 开发经验，两年以上 Web 开发经验（如果你有 3 年以上其他语言的 Web 开发经验，Ruby 的要求可以忽略）；
  • 熟悉 HTML, Javascript, CSS，能自己搞定前端开发的工作，并能处理好浏览器兼容问题；
  • 熟悉 MySQL；
  • 熟悉 Git;
  • 能熟练在 Linux 或 Mac 环境工作；
  • 当然有经验的优先；

工作内容

  • 全职
  • 根据需求完善网站的功能

薪资

  • 新手 8K，老手 15K
  • 工作设备有 Mac, 全新 iMac

工作地点：

  • 杭州西溪路浙大科技园 C 栋 606

简历请投递： huacnlee@gmail.com
