[欧洲] Chinese speaker Ruby/JS/Web Developer - 欧洲工作机会，有兴趣的不要错过

招聘 • sherry • 于发布 • 最后由 sherry 于回复 • 5262次阅读

工作地点在欧洲。非猎头，直招。非诚勿扰

亲们注意，由于是在欧洲工作，所以要求英文沟通无障碍，所以职位要求，我就直接贴英文了，看不明白的可以直接绕过。薪资福利在文章底部以中文贴出。

联系方式：

QQ:616512942

招聘人数：1人

职位名称：Chinese speaker Ruby/JS/Web Developer

As a web developer you have a strong background in web development with expertise in Ruby and Javascript. You are passionate about modern web technologies and are able to work with a distributed agile team in a fast paced environment.

You are:

  • Experienced with web development on both mobile and desktop: you are an expert in Ruby and Javascript frameworks (RoR, Backbone, jQuery, ...) and have experience in developing rich client applications using modern web technologies across both web and mobile
  • A good communicator: you communicate well with both technical and non-technical staff, across distributed teams and local departments
  • A keen learner: you are able to adapt to new, complex development environments and stay up-to-date with the latest technologies and trends

You will:

  • Be part of an agile development team
  • Deliver implementations in accordance with coding standards and conventions
  • Architect, implement, and maintain Ruby on Rails and JavaScript based components
  • Rapidly solve problems and bugs
  • Play a key role in the full development cycle and direction of feature sets

薪资待遇：

工作地点：欧洲
薪资：税前英镑35K-43K一年（取决于你的能力）
奖金：取决于公司业绩，业绩好时，公司根据各人情况发放奖金
工作时间：8小时（包括午饭时间）每天早上10：00之前到公司即可，达到8小时即可离开
带薪假期：32天
带薪病假：20天？- 这个不太记得了，偶不是HR的，不过应该这个上下，女性另有产假。还有一些其他假期就不例举了，比如，丧葬假等

福利：

  1. 转正后，公司提供养老保险，员工自己无需出钱，公司给每月帮供基本工资的4%-5%，员工离开公司后，可以直接提取，免税。
  2. 医疗免费
  3. 每天3镑的伙食补贴（当天不用作废，包括周六周日，公司自己有餐厅）

迁移补贴:

  1. 新员工，有三个月的300-500镑的住房补贴，三个月后取消。
  2. 所有签证费用公司全额报销
  3. 公司订机票
  4. 公司提供上限为250镑的行李托运及机场中转时产生的费用，到公司后报销
  5. 所有工作签证等手续费用由公司支付

说明：

  • 大家对为什么要加Chinese Speaker有疑问，这里统一解释一下：

工作主要是对欧美网站的，但是偶尔会有做中文网站的项目，所以要懂中文的。而且如果是只要英文的，公司完全不用在中国找人过去了。

而且在办理工作签证的时候，如果前面不加 Chinese Speaker，政府是不会批工作签证的，因为如果说英文的人能做这份工作，政府是不会让公司请中国人的。

  • 有人问，为什么不在欧洲招，这里也说一下

公司有通过猎头在全球招人，因为猎头本身也是在欧洲的，又懂中文又懂RUBY的不多。这们公司加我就两个中国人，所以HR就请我和我另一同事在国内给关注一下有没有想过来的咯。反正给猎头中介费也是给，从中国招人relocate过来也是给，所以就双管其下咯。先到先得，机会不等人哦

  • 我想说一下简历的重要性

陆陆续续收到一些简历，其实我不是猎头，直接就把简历转交上去就可以，结果完全不用我管。但是我还是要面子的，简历上去了，写的太随意，我的面子都是挂不住的。
我想对那些粗制滥造的简历制造人说，如果你们是用人单位，看到这样的简历，你们会给面试机会给他们吗？
是不是真心想申请一份工作，其实从简历的制作上就可以看出来。我真想不点名的爆几个简历的笑料，不过我想想还是算了，给你们留点面子
我真心希望在以后你们申请工作的时候，用点心，因为往往这点用心就能让你在很多申请人中脱颖而出
