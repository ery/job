本公司为互联网旅游平台因为业务发展需要诚聘IOS开发主管，Android开发主管，ui，ruby,前端工程师

技术部的汉子热情专业，市场部的妹子善良美丽，工作环境有湖有树有菜园，负离子丰富有益身体健康。此外还管吃管住，让你在工作之余没有生活的烦忧。

别提示：为您的远大前程着想，请认真阅读本要求。

职位一：iOS开发主管

岗位说明：
1、负责iPhone/iPad平台下应用的开发
2、和产品经理、测试等进行沟通，设计并实现具体模块

要求：
1、2年以上工作经验，至少1年以上 ios 开发经验
2、熟悉iOS完整开发流程，熟悉iOS UI、数据库、网络、推送、多媒体等相关功能开发
3、熟悉各种iOS第三方库
4、热爱设计编写程序，对IT行业充满激情，有独立的用代码解决问题的能力，对软件产品有强烈的责任心
5、熟悉iOS应用开发框架，能独立开发高性能的iOS应用

职位二：Android项目经理

要求：
1、负责Android平台上的客户端的程序设计、开发和维护
2、负责完成基于客户端的各项文档编写
3、根据项目需要快速学习并掌握技术

任职要求：
1、1年以上开发经验
2、具备扎实的Java编程基础
3、熟悉Android SDK，
4、熟悉Android下的网络通信机制，具有丰富的UI开发经验
5、良好的编码习惯，对软件性能优化、内存优化有一定的经验
6、具备一个或二个以上的成熟的应用项目，有过上架或正式交付使用的作品
7、有良好的团队精神、敬业精神和沟通协调能力。

职位三：ui工程师

职责
1，负责公司基于浏览器的产品 UI 设计 和客户端UI设计
2，规划产品的视觉风格
3，有效平衡网站的易用性和美观性
4，把握设计趋势，提出新的设计和创意策略
5，和产品及开发团队一起进行前期的概念及原型的设计

要求：
1、精通 PS / AI 等设计软件，
2、善于沟通和用视觉来表达想法，能有效地将想法表现为设计方案
3、一年以上互联网行业 UI 设计相关工作经验

职位四：前端工程师

职责；
1，Sinatra + Mongoid 开发一款初创产品；
2，Bootstrap 、 jQuery 必须没的说~
3，我们目前是 UI + Rubyist 简配~
4，深刻理解Web标准，在可用性、可访问性、浏览器兼容性等方面有实践经验；
有设计作品，请随简历一起附上作品或可查看的链接地址

职位五：Ruby工程师

职责：
1，ruby, mongoid, 熟悉sinatra框架或者rails
2，会jQuery和bootstrap加分
