[北京] Jing.fm 找 Android 的高级攻城狮，运维攻城狮，Ruby 攻城狮

招聘 • chenillen • 于发布 • 最后由 wcp1231 于回复 • 1295次阅读

Jing.fm開創了全新的音樂搜索概念，在短短上線1年左右的時間已經在互聯網中引起了巨大的反響。

目前产品有基于HTML5的Web App的http://jing.fm http://jing.fm/beta/

IOS ： https://itunes.apple.com/cn/app/jing/id581886398?mt=8
Android版本： https://play.google.com/store/apps/details?id=com.jingfm&hl=en

我们希望你能认同：

用户体验是一个伟大产品的核心和最有效竞争力。
公司小而精，做的事情伟大
我们的产品可以改变用户的听音乐的习惯

我们希望你是一个：

野人范的音乐爱好者
对细节的孜孜以求的完美主义者，包括代码书写，字体用法，图片的加载等
不仅仅是一个编码器，你也是用户也是产品

你将会在这样的环境下工作：

开放，年轻，富有激情的办公环境
你有一群对新技术狂热的同学和你一起讨论
你还有一群对音乐痴迷的同学跟你分享音乐
时不时的还有各种音乐派对

如果你是以下岗位的人才，不要犹豫，立刻发简历给 a@jing.fm 。
请在邮件标题注明应聘的职位。

——————————————————————————————

Android高级攻城獅

  1. 能够独立编写 Android 应用，有完整作品（Google Play）加分
  2. 精通Android UI框架，并且掌握其他UI框架、效果等
  3. 有多媒体应用开发的经验，流媒体服务应用优先
  4. 熟练使用 Git ，以及在 GitHub 上有帐号和项目
  5. 使用Mac进行开发的同学，优先。
  6. 会翻墙，有独立的思维

——————————————————————————————

运维攻城獅
1. 認同命令行就是生命，重要性遠遠超過GUI
2. 集羣服務器和虛擬化是以後的方向
3. 熟练掌握bash/shell/zsh/tsh其中之一
4. 熟练掌握LVS，mysql主从、redis配置优化等
5. 流媒体服务器配置，CDN优化等经验

** 有多IDC架构运维经验者，加分
** 熟悉puppet、chef等统一管理服务器经验
** 掌握python/perl/ruby三者之一脚本语言优先
** Mac 用戶，加分

——————————————————————————————

Ruby攻城獅

——————————————————————————————

  1. 喜歡Ruby及Ruby社區的氣氛的攻城獅
  2. 精通Rails，但不僅限於Rails框架，例如sinatra、grape、goliath
  3. 精通Rspec、TestUnit等測試框架，是一個TDD/BDD的忠實擁躉
  4. 在Github有基於Ruby開源項目，對數據分析、處理有研究
  5. Mac 用戶
  6. 喜歡折騰、各種refactor、喜歡新東西

我們對攻城獅沒有工作年限，只要你夠有水品，Jing都會為你『倒履相迎』。
