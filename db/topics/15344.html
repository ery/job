[上海] 银联智惠 诚聘 Ruby 研发工程师

招聘 • rissa • 于发布 • 最后由 xstmjh 于回复 • 1314次阅读

亲们~目前招聘初级Ruby工程师（2人），高级Ruby工程师（2人）

岗位职责:
负责公司核心产品的架构设计及功能开发 ；
参与产品的需求调研和需求分析 ；
编制并实施项目概要设计、详细设计 ；
完成系统框架的搭建和核心代码的实现 ；
负责解决开发过程中的技术问题 ；
完成单元测试、文档编写、版本控制等开发工作 ；
与前端工程师紧密配合完成功能实现 。

岗位要求:
1.精通ruby或者java语言，精通网络编程并有高并发系统的设计开发经验 ;
2.熟悉Mysql数据库；
3.熟悉NoSQL架构，有主流分布式缓存系统开发设计经验；
4.熟练使用Linux，熟悉shell/ruby/python脚本；
5.知识面广、善于学习。
具备以下条件者优先
热爱软件开发设计行业；
对算法的设计和分析具有一定的了解；
有良好的代码书写和编程习惯；
具有丰富的java或ruby架构设计经验优先。

公司介绍：
银联智惠是中国银联旗下子公司，成立于2012年，核心团队曾在ORACLE、Morgan Stanley、阿里巴巴、腾讯、百度、盛大等国内外知名企业担任重要职务，拥有成熟的公司治理经验，市场/销售经验，大数据挖掘核心技术，并拥有多项软件著作权。
公司定位为开放式大数据平台，致力于大数据的整合、挖掘、应用，为零售和金融行业提供深度数据服务，提升客户企业运作效率，发展创新业务模式。
公司依托银联强大的跨行转接交易数据资源，采集并整合银行、运营商、互联网等来源的数据，建设统一的大数据基础平台；在此基础上，采用数据挖掘核心技术，精心打造银联智惠大数据模型，勾勒出全国8亿持卡人的画像及500万商户的特征，形成公司的核心资产；同时，开放数据调用接口，自主或合作开发基于数据模型的应用服务，实现大数据的商业价值。

公司网站：www.unionpaysmart.com
公司地址：上海浦东张江龙东大道3000号
（ 考虑到员工上下班更便捷以及公司人员的逐渐增多，年末公司即将搬往世纪大道附近）

1.五险一金、补充公积金

2.在我公司工作满1-2年，5天年假；2-5年10天年假
3.5天带薪病假
4.每年一次常规体检
5.旅游和团队活动
6.节日生日的问候
7.其他（例如：公司女员工怀孕，提供专用防辐射服等等）

团队介绍：
团队精通于技术，着迷于数据，崇尚快速、自由、创意至上的工作氛围。公司兼备国内独家的强势资源，亦有海归创业公司的活力和激情。纯技术氛围，不存在你争我夺的竞争，而是相互分享，互相帮助团队。热衷于技术分享，每天 Code Review，每一位同事都成长飞快。
如果你是技术怪才；
如果你和我们一样有激情、有梦想；
如果你对新兴行业有着近乎偏执的热情；
我们会以自由宽松的工作环境、快速成长的业务和未来的无限前景作为交换；
期待你的加入，一起努力，迎接挑战，共同成长；
诚邀具有同样梦想的你加盟！
