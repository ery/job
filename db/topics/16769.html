注：此职位可以是高级工程师，也可以是首席工程师。

首席工程师：

岗位职责：

  1. 在技术和运营上负责一个或多个产品组件、关键应用的架构改造、升级、研究与开发；

  2. 关注互联网热点技术（广告、大数据、Web前端等）的发展方向，敢于提出、评估并应用新技术，带领团队开发新的工具，帮助提高现有系统的效率、降低运营成本、提高可靠性；

  3. 作为技术专家，解决系统开发中的疑难问题。在产品特性与架构演化之间做出良好的技术决策、平衡与取舍；

  4. 能够与产品经理、管理团队进行良好地沟通合作，确保项目按时、保质完成，培养若干有潜质的工程师。

任职资格：

  1. 本科以上学历，计算机相关专业，本科需具有8年以上、硕士需具有6年以上工作经验，其中至少4年以上的大型互联网系统开发经验，2年以上的技术主管经验；

  2. 掌握多种主流程序设计语言。有大型Web应用开发和架构经验，熟悉主流Web应用框架，精通Ruby或Java；

  3. 熟悉Linux操作系统、网络和关系型数据库的应用开发与优化；

  4. 对设计模式、软件工程、用户体验等有较深入的理解；

  5. 擅长组织、沟通及协调各方资源，具备一定的技术团队管理经验；

  6. 可以熟练地使用英文工作，适应外企环境。

Sr Software Engineer:

工作内容：

  1. Build Enterprise level video monetization B2B web application.

  2. Operate the system, closely monitor the system with OPS team, adjust and optimize system to meet high Service Level Agreement.

  3. Design and build Web UI foundation to support application development.

  4. Make day-to-day tech decision to grow our global business.

应聘资格：

  1. Strong interest on web technologies, e.g. Ruby on Rails/Ajax/JS/HTML/CSS/Open API/SSO etc.

  2. Solid knowledge on Linux, network and MySQL.

  3. Experiences on building multi-tenancy system would be a great PLUS!

  4. Good English communication.

  5. Bachelor or Master degree in Computer Science related major.
