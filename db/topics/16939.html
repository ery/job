　　如果你想体验创业旅程的刺激又不想承担太大的风险；
　　如果你想参与团队和产品的方方面面又想在专业上有所突破；

关于我们

　　我们这个团队脱胎于于东方网力科技股份有限公司
　　经过一段时间的内部的孵化，现在开始进入独立运作阶段
　　我们相信：
　　工作不仅仅是人们赖以生存的手段，还是让生命变得更美好的途径
　　我们的目标：
　　打造让自己满意，让朋友称赞，有很多人用的，能真正有助于人们工作的app；
　　我们的愿景：
　　人们可以把大部分时间和精力投放在创造性的事情上，协调和组织自然而然且随时随地，每个人都能找到自己的准确坐标并将自己所擅长的价值最大化。
　　产品暂未对外开放，传送门：http://www.yunfis.com
　　
职位
　　Ruby工程师2名
　　岗位职责
　　后端业务的处理,为前端提供稳定的API.
　　岗位要求
　　1. 3年以上工作经验，计算机基础扎实, 熟悉常见数据结构和算法, 良好的逻辑思维.
　　2. 熟练使用 ruby on rails, mysql, 并能够手写出常见查询.
　　3. 使用 linux 为工作环境, git 版本控制.
　　3. 有github主页或者blog, 积极参加开源项目. 有Javascript, C, Java 相关开发经验.
　　5. 有企业应用或大型应用开发经验.沟通能力好，良好的团队协作意识, 有责任感.
　　6. 代码简洁干净,追求完美. 爱读书,爱写代码,爱学习.
　　
待遇
　　年薪12w~30w，期权，五险一金，……（其他个性化要求可面议）
　　
地点
　　北京回龙观龙冠商务中心（万意百货对面，北边的兄弟们不用忍受通勤之苦啦）
