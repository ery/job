公司简介

英国互联网移动开发公司Zenpow Ltd成都分部招聘一名Ruby工程师。公司主要业务是做外包，也在做创业产品的计划中。公司建立2008于英国伦敦，做开发多年有长期靠谱的客户。

职位

  • 一名Ruby on Rails工程师

希望你

  • 有8个月以上的Ruby经验
  • 知道如何写clean code
  • 熟悉git, linux, rspec
  • 有不错的英文阅读能力
  • 一个积极的学习态度
  • 善于交流
  • iOS开发经验(用Objective C或Rubymotion)是优势

工作环境

  • 良好的办公环境：空调，除湿器，空气净化器
  • 办公室在成都市内 - 不用跑到远远的软件园去上班
  • 办公时间：9点到下午 5：30 - 有活力
  • 不加班
  • 免费提供健康home cooked午餐 - 可点喜欢的饭菜
  • 国家假期 + 每年10天自选假日
  • 可以选择一部分时间在家办公
  • 公司有很多国外精彩的原版Ruby/Rails视频教材（RailsCasts Pro, Peepcode, Destroy All Software 等等）- 帮你更快的转到高手

待遇

  • 月薪6-8.5k （具体面谈）
  • 年奖
  • 项目奖励

联系方式

请填表格 http://goo.gl/RCdCEF (最好方式；应该不需要翻墙).
或简历投递 zac at zenpow dot com

地址: 四川省成都市玉林北街一号玉林品上（省体育馆附近）
