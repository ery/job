[Telecommute] 加拿大创业公司寻Web前端（兼顾Rails）技术主管（在家工作10K+）

招聘 • focuser • 于发布 • 最后由 focuser 于回复 • 2728次阅读

"The best way to predict the future is to invent it" -- Alan Kay

大家好，我是一个在加拿大创业的华人开发者，希望结识跟我们有着同样理想、渴望改变世界、创造未来个人计算模式的朋友。我们是积木实验室(http://jimulabs.com)，我们坚信，软件应用的开发，应该像儿时搭积木一样轻松自然，每一个人都应该能够量体裁衣，创造出完全符合自己需要的应用。

基于这个理念，我们正在研发一款移动应用创建平台，让人们不用写代码也可以制作出高品质的原生移动应用。这个领域现有的工具包括Google的App Inventor（现在已转至MIT），以及满大街的基于PhoneGap等Web技术的App maker，然而，采用这些工具只能制作出一些功能有限、体验差劲并且难以后续定制的应用。我们相信基于积木的代码生成技术（网站上有youtube演示视频），我们能够做得更好！

如果您是一位通晓Web前端的高手，对用户体验有独到见解，追求细节完美，我真诚的邀请您加入我们的团队。系统的开发刚刚开始，因此对其架构您有相当的自由性（做新系统是程序员最幸福的时刻，这我理解）。

为便于双方增进了解，项目分阶段进行，第一期两个月，在家工作，弹性时间。根据具体工作量，如果是全职，每月报酬不少于10k。如果是兼职，每小时报酬不少于70元。

要求：
- CoffeeScript, Sass, HTML5, CSS3
- 有开发单页Web应用开发经验（single page web app）
- RESTful API, MVC
- jQuery, Ember.js, jsPlumb
- Jasmine、qUnit等BDD/TDD框架
- Ruby on Rails
- Git
- Linux或Mac
- 良好的沟通能力，英语读写流利

加分：
- 有一线运维经验，部署过实际RoR项目
- 有开源作品
- 有Android/iOS开发经验
- 能用英语进行口头交流（便于提高和团队的交流效率）

请将对此问题的解答、简历、技术博客网址、Github、Stackoverflow账号、以往作品等发送至newblood@jimulabs.com，并告知为何您是这个职位的不二人选。

谢谢！
