#8楼 @sztonny 谢谢你的建议。

顺便，看了一下小恩爱的首页，发表观点如下。特别声明，以下纯属个人观点：

  1. 使用 HTML5 吧，Google 都已经开始使用了，所以不存在什么顾虑；
  2. id 的命名全部小写，多词连接用 “-”。小恩爱现在的 id 命名方式，让我觉得这位前辈是 JS 达人；
  3. 页面字体：字体的定义尽量不要用汉字；
  4. 样式标中大量使用中文注释，但是没有定义 @charset；
  5. 某些 JS 还是放在页面底部吧，像 Google Analytics 之类的；
  6. jQuery 建议使用 Google CDN 上的，然后会滚到本地的；
  7. LOGO 的链接 text 居然没有，这和新浪微薄一样错了，使用 image replace 就好了嘛；登录按钮链接同理；
  8. .banner 这个就用 img 最好，ir 就没必要了；
  9. 首页的 h1 居然是“一切源于沟通和互动”，而不是站点名？
  10. “点击申请” 的 inline javascript；
  11. CNZZ 统计 display:none，这一点很赞。
