[北京] 酷传招聘 Ruby on Rails 工程师

招聘 • liweijie • 于发布 • 最后由 greatghoul 于回复 • 264次阅读

酷传是一个平台型的移动互联网公司。

酷传主要方向是为移动互联网开发者提供服务，帮他们更便捷，低成本的推广应用产品，获取更多用户量，满足最为刚性的开发者需求。

公司致力于为中国开发者提供专业的移动应用分发工具、实用组件以及平台式推广服务，积极的帮助开发者推广他们的产品。

酷传也是未来移动互联网的数据中心，在这里你能看到行业走势，潮起潮落。团队成员全部来自百度，创新工场，有着丰富的移动互联网项目经验以及一线实战经验，这多半是一生没有几次的机会，加入酷传你会有不一样的收获。

我们在寻找的职位 Ruby on Rails工程师

成长空间：
1. 参与产品的设计，开发，测试，维护等完整的开发流程
2. 独立负责酷传新产品的的开发

要求：
1. 熟悉ruby，具备良好的编程基础，追求代码质量；
2. 1年以上Ruby on rails开发经验；
3. 熟悉 Linux，Mysql，git；
4. 对 Web 前端 (HTML/CSS/Javascript) 有基本的了解；
5. 积极，主动，责任心和团队合作意识，较好的学习能力

加分：
1. 有redis、mongodb经验
2. 有服务器运维经验
3. 了解函数式编程，比如lisp，scheme，clojure

我们能提供的：
轻松的团队氛围，我们痛恨办公室政治等一切阻碍生产力和个人发挥的东西，你懂得。（甄嬛传太可怕鸟~）
假如你是新人，恭喜你，你来到了一个真正做事的公司，能切身感受到自己的成长，学得最潮的技术，参加行业最牛的分享，帮你养成良好的编程习惯。

薪酬待遇： 8K—15K

联系邮箱：简历发送至 hr@coolchuan.com

公司地址：北京 地铁10号线 健德门站（出站即到）
