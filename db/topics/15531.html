工作地点: 徐家汇

要求：

  1. 熟悉ruby，具备良好的编程基础，追求代码质量；
  2. 1年以上Ruby on rails开发经验；
  3. 熟悉 Linux，Mysql，git；
  4. 对 Web 前端 (HTML/CSS/Javascript) 有基本的了解；
  5. 较好的学习能力
  6. 积极，主动，责任心和团队合作意识，能有条理安排工作

具备以下条件者优先

  1. 有redis、mongodb经验
  2. 有服务器运维经验的
  3. 有Go/Scala/Clojure等其他编程语言经验的
  4. 有客户端应用程序开发经验的

薪酬待遇： 8K—20K

对该职位感兴趣的同学请邮件请联系caoli@rongyi.com

附公司简介

上海容易网电子商务有限公司简介

上海容易网电子商务有限公司，总部位于上海，在深圳、北京设有子公司，业务分布于国内主要的25个主流城市。

容易网成立于2012年4月，注册资本1000万元，北京子公司注册资本1000万元，深圳子公司注册资本500万元。容易网所在集团注册资本3000万元。

容易网专注于为线下零售业提供智能化导购、展示的技术和营销整体解决方案，为各类综合购物中心、零售业品牌商户提供基于店内展示和人机互动的O2O立体式服务，有效盘活客流、提升平效；同时致力于为消费者创造更为丰富、便利的消费体验和更高品质、品牌化的服务保障。

公司创始团队汇集投资、管理运营、互联网技术、市场营销等各行业的资深人士。产品和服务建立于消费资讯网站、手机应用、商场终端机三端媒介之上，并实现有机组合、立体覆盖，帮助线下零售业主与消费者实现在购物实体环境中的即时性体验、互动和分享，让消费者的购物过程更高效、更愉悦，让线下零售业主赢得更活跃的消费行为。

容易网电子服务体系包含：容易逛智能终端导购机、容易逛手机APP、容易购智能终端电子货架、rongyi.com容易网导购门户等。

公司创始人/CEO曾任百度总裁顾问、分众传媒首席运营官、航美传媒首席战略顾问、凤凰卫视华东区总裁，曾获2007年中国十大经济女性年度人物、2013年中国十大经济女性年度人物特别荣誉大奖。

容易网横跨多种平台，拥有在互联网、移动互联网、零售业、广告业有多年经验的创业团队，现正大规模在全国拓展，热忱期待你的加入。
