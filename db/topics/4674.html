[上海] 八月上海诚聘 Ruby on Rails 兼职一名

招聘 • shephie • 于发布 • 最后由 gazeldx 于回复 • 1158次阅读

We would like to partner up with someone for the development of a Rails-based project.

The project we have in mind is an event management system, for which there will be an administration site for the event organizers and a public facing microsite for the event attendees.

On the microsite, invited guests will be able to log in (by scanning a QR code or entering a personal URL) and register for the event at a particular date and location, as well as answer a questionnaire and book activities during the event.

On the administration site, the event organizers will be able to manage the event dates and locations, as well as define and manage activities. The administration site will also provide the ability to import lists of guests (email, name) the list of invited guests, generate QR codes for printing on personal invitations and check up on guest registration status.

The tasks would be assisting us in building and refactoring the data model (ActiveRecord & MySQL), building the administrative site (we are considering the use Active Admin) as well as the public-facing microsite. For this purppose, it would be ideal for us to work with someone who is comfortable to work on the backend part of web applications.

We are based in Shanghai, and use Git and Redmine to manage our source code and project. While we prefer to work with someone who can come to our office, it is also possible to work remotely. The deadline for this project is August 31.

Looking forward to hear from you.

各位Ruby高手，大家好！

我司是一家专注于多媒体开发制作的德国公司，中国办事处在上海。现有一基于Rail开发的项目，需有才有闲的大侠能在八月鼎力相助。

这是一个活动管理系统开发的项目，会有一个面对大众的活动网站和一个面向活动组织方的管理员后台。

在活动网站上，用户可以通过扫描二维码登陆，输入个人信息、查询活动信息、提交问卷或预约感兴趣的活动。

在后台，管理员可以查看用户信息、批复活动申请、更新活动信息；他们还可以导入潜在用户信息，系统会为每个用户自动生成便于身份识别的二维码，管理员可以查询活动预约的情况。

我们需要兼职大侠帮助的部分是帮我们搭建和重构数据模型（ActiveRecord & MySQL），搭建后台（我们考虑使用Active Admin）和前台的活动网站。出于这样的需求，我们希望能与熟悉Web应用后台开发的大侠合作。

我们的办公室在上海，用Git和Redmine管理代码和项目。最好大侠能来我们办公室一同齐头并进，当然如果条件所限，我们也可以远程合作。项目已经启动，会在8月31日结束，所以哪位大侠八月有空，不妨一试。薪资面谈，好说！
