1. 参与迭代执行并完成功能开发；
2. 分析需求文档，并与产品人员沟通；
3. 评估开发计划执行情况，提交测试包给测试组进行测试；
4. 与测试组产品组相关人员沟通，获取BUG文档，分析BUG并制定修改计划；

任职资格：
1. 计算机及相关专业本科以上学历；
2. 熟练掌握java语言，熟悉面向对象编程设计。
3. 熟悉Android框架原理及Activity生命周期、电源管理等，精通 Android UI布局，熟练使用SQLite数据库
4. 精通Socket通信，深入理解TCP/IP和HTTP通讯过程。
5. 具有责任感和荣誉感，踏实稳重；能独立钻研解决技术问题。
6. 有NDK开发经验者优先。

岗位2：IOS手机开发工程师

工作职责:

  1. 负责IOS移动应用程序的项目开发；
  2. 依据项目需求，基于Apple IOS平台进行移动产品的设计与开发；
  3. 评估开发计划执行情况，提交测试包给测试组进行测试；
  4. 与测试组产品组相关人员沟通，获取BUG文档，分析BUG并制定修改计划；
  5. 编写、整理技术开发文档，服务器协议文档；

技能要求：

1.具有二年以上IOS平台应用开发经验；
2.有扎实的C/C++/Objective-C基础，了解应用基本算法/数据结构，了解基础设计模式；
3.熟悉IOS SDK开发环境及相关开发工具；
4.熟悉IOS平台相关通讯架构，精通Socket通信，深入理解TCP/IP和HTTP通讯过程；
5.熟悉Json，Sqlite等在IOS平台中的应用；
6.具有良好编码风格，有较强的独立工作能力和团队合作精神；
7.思路清晰，思维敏捷，快速的学习能力；具备良好的沟通能力和团队合作精神；
8.有在app store上发布软件者优先。

任职资格：
1. 计算机相关专业大学本科以上；
2. 有较强的团队协作意识及独立解决技术问题的能力；
3. 无障碍阅读相关专业英文资料，愿意学习新的移动平台技术；
4. 具备很强的学习能力、沟通能力和团队工作精神。

荔枝FM是即将推出的手机电台软件（如果你使用的苹果设备已经越狱，可以在91手机助手或PP助手搜索“荔枝FM”安装体验了罗~~）“荔枝FM”的理想是“重新发明电台”，公司目前正在进行“荔枝FM”电台项目，目标是帮助人们在手机上实现人人都可以做主播的梦想。

公司2010年完成A轮融资，获得了著名VC经纬创投的投资。2011年完成B轮融资，获得著名VC晨兴创投以及经纬创投的联合投资。迅雷、小米手机、YY语音、UC浏览器等优秀的创业公司都是 和我们有相同投资方的兄弟公司。

荔枝FM现有员工20多人，绝大部分是技术、产品人员，来源于华为、爱立信等公司，是一个人数不多但精干的团队，团队成员大部分有提供过千万级别用户的 互联网服务经验。
