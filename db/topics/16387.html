说说优点

  • 有创业公司氛围，公司结构扁平，没有title，有啥想法可以跟任何人聊，都很热心的。
  • 公司刚拿到千万美元的投资，资金方面是没问题的。
  • 出境自助游的增长势头迅猛，行业前景可期。
  • 午饭有阿姨做，平心而论还不错，花样很多。
  • 如果你喜欢旅行，这里会接触到很多旅行达人

缺点

  • 和大多数创业公司一样，很辛苦，初期工作时间是 5x12

关于技术

这方面的自由度很高，网站运作了快一年，但有很多方面有待完善，目前涉及的技术是 Grunt/Jade/Stylus/AngularJS/RequireJS 等，减少了重复劳动，开发体验提升明显。

我们还有好多事没做好，也就意味着你除了正常开发以外会接触到以下的工作，如果未来的同学有已下经验也希望能推进这些项目

  • 基础 UI 库，网站已逐步稳定，将会着手整理一套，jade,stylus 结合 angular 的directive 相信会很有趣
  • 测试，之前没这方面经验，希望有经验的同学来能推动
  • 更好的文档和demo演示
  • 前端环境自动化，目前做的还不错，但还有很多地方可以改进
  • 环境发布，让部署更有体验
  • node.js 可以接手一部分后台的工作，让整个项目更有效率
  • 和项目结合紧密的脚手架，让开发更高效
  • 更好的项目架构，会考虑类似 bower，component
  • 动画和各种css3的效果，这块确实经验少
  • 移动端，响应式，后面这方面是重心
  • 页面优化
  • 安全

未来的同学应该是

  1. 对前端有浓厚兴趣，html/css/js 具备前端应有的技术素养
  2. 能接受各种不同观点，做技术每个人可能都有自己不一样的理解，但希望都有开放的心态。
  3. 知其然知其所以然，解决问题是很简单的，探究问题原因才治根
  4. 有DRY原则，不重复工作，能用工具和脚本解决的问题不会手工做第二次
  5. Google，Github，StackOverflow，日常必备，*nix 下开发没压力

如果以上条件不满足，那希望你有何别人不一样的特质，或者技术
