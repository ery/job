距离上一次发这个广告已经是8个月过去了，在这8个月里面Vida团队做了什么？

  • Vida用户突破5百万了！
  • Vida团队搬家了，搬出了温馨的三室一厅，进驻了延安高架边的写字楼，虽然没有了知性阿姨的烧的晚饭，但是晚餐补贴+定西路，吃的东西绝不缺少。

  • 更重要的是，Vida扩产品线了，围绕着XXX为中心，扩充了更加Sexy的产品，所以更加需要靠谱的攻城师/架构师。

    欢迎大家加入到靠谱的创业团队！

创业公司招人是一个公认的难题，其实主要问题是三点：

 -  产品有没有前途？

 -  团队是不是靠谱？

 -  资金支持是否足够？

VIDA的回答：

 -   产品是否靠谱，靠谱的人自己会判断，请到各大iPhone/Android市场搜索“VIDA”下载使用体验。VIDA作为手机UGC照片和微视频社区，正式发布半年多来虽然发展成为国内数一数二的富媒体社区，需要改进的地方很多，欢迎提建议。


 -  团队是否靠谱，来看看就知道，没有最好，只有合适的团队。现在各种背景的人都有，啥google/microsoft/IBM前员工都混在一起。有一把年纪的，70后互联网资深老鸟，年轻的，90后ACM金牌得主，大家在一起只是为了做一款靠谱的产品。


 -  资金支持？已经两轮VC投资了，最新的一轮刚刚落实，有钱缺人！

所以我们求靠谱攻城师！！！

系统架构师

职位描述

* 设计、构建，维护高可用高负载高安全性的服务器集群
* 优化提高服务器反应速度

要求

  • 精通 Linux 与分布式系统开发，熟悉 bash 以及至少一门动态语言
  • 熟悉数据库设计，了解表结构性能优化设计
  • 了解高并发，大数量的数据请求处理方式

Web 攻城师

职位描述
* 开发、维护VIDA的各web/mobile web产品
* 优化VIDA产品性能

要求

* 熟悉 python、ruby 等动态语言，精通 Rails
* 熟悉前端技术技能，熟悉 jQuery 等前端框架
* 熟悉各浏览器差异，了解 Web 标准
* 熟悉互联网产品开发流程
