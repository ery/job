4.18更新

目前已经招到前端和后端工程师了，感谢各位支持。欢迎iOS工程师和设计师乱入！

记得Ruby-China第一张招聘贴就是我们发的，http://ruby-china.org/topics/90，转眼一年过去了，不知道大家是否安好。我们公司不但没有倒，反而更快速地在发展。目前已经在音乐行业站稳脚跟，正准备冲击下一个里程碑了。在此再次感谢大家的支持和关注，请帮忙转发和通告我们的这一次招募。

我们做什么 ?

我们相信音乐应该有它的价值，因此专注于服务音乐人、音乐行业，希望可以推动音乐产业的发展，改善音乐人的生活。更多公司介绍可以上公司网站查阅：www.gaiamagic.com

音乐人攻略，音乐行业的36氪+虎嗅。

合拍网 ，做音乐人常用的社交性工具，例如照相的instagram、编程的Github。

友演出，Web和手机APP，音乐演出信息的去哪儿。

创始人

我是创始人马龙，87年，初中就开始做一些球鞋衣服生意，但是后来不希望埋没在琐碎的高买低卖里面，所以立志做一番事业。两年前毕业即创业，创业时不懂编程，不懂设计，所有东西都依靠团队。因此很感恩也很重视公司人才。

身份多变，除了商业、互联网、音乐之外还对电影、时尚、美食有涉猎。口味方面可以看 本人豆瓣匹配一下，东西加得不全。微博是@馬龍GaiaMagic，欢迎勾搭。

现状如何

团队：团队总部坐落在广东佛山，成员超过十人，非常完整。合作支持远程协作和兼职，其他成员分布广州、北京、广西等地。

收入：今年会实现盈利

行业：音乐领域现在目前正在往正版化、数字化的方向推动，复苏已经是板板上的事情了。

招聘职位

关于工程师

技术栈：

  • Ruby && Rails && RSpec
  • MongoDB && Mongoid
  • Redis
  • JavaScript && CoffeeScript && jQuery
  • SASS
  • Git （我们不用 Git-Flow，用基于 Github Pull Request 的 Flow ）
  • Mac || Linux

加分项：

  • Vim || Emacs
  • AngularJS 或者其它相似的前端 MVC 框架
  • 对其他语言也有了解，如 Python、PHP 等
  • 参与过开源项目并有所贡献
  • 有独立的技术 Blog

关于设计师

职责：

  • 设计APP、Web、Banner、Logo、印刷品
  • 产品的交互效果和逻辑

要求：

  • 在细节和简单中找平衡，做出让人情不自禁Wow的设计
  • 了解 Web 和移动端的交互并能有所创新，让产品用起来更自然，交互更不露痕迹
  • 思维能力强，对不同设计对象都能瞬间能找到重点

加分项：

  • 有 Dribbble 账号
  • 能手绘，有美学基础
  • 了解前端

除此之外，还有iOS工程师、运营总监、音乐人攻略编辑等岗位招聘，可上www.gaiamagic.com查看

福利与待遇

福利：

  • 无限提供各式饮料、星巴克下午茶、百果园水果
  • 苹果电脑、大屏幕、人体力学凳、机械键盘配置
  • 粤菜大神每天做住家饭，煲靓汤
  • 弹性上班时间
  • 外地成员投奔总部还将安排住宿和阿姨搞卫生做饭
  • GaiaMagic图书馆，加上每月可享书籍报销

待遇：底薪+奖金+期权

  • 底薪根据岗位而定，骨干成员最少6K，上不封顶
  • 每季度发团队奖金
  • 骨干成员入职半年即获最少十万期权
  • 六个月一次加薪
