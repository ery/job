#39楼 @xautjzd 我给你看另外一个类似的例子吧
首先 routes.rb

  get ':year/:month/:day/:slug' => 'posts#show', :constraints => {
                      :year     => /(19|20)\d{2}/,
                      :month    => /[01]?\d{2}/,
                      :day      => /[0-3]?\d{2}/
                    }

然后对应的 posts#show 里面，可以 find_by_slug(params[:slug])

另外还要自己写 URL 和 PATH 的方法，类似

  def post_pretty_path(post)
    "/#{post.created_at.strftime "%Y/%m/%d"}/#{post.slug}"
  end

  def post_pretty_url(post)
      root_url + "#{post.created_at.strftime "%Y/%m/%d"}/#{post.slug}"
  end

这是我之前写 abc.com/yyyy/mm/dd/postname 这种 URL 的例子
