Web前端工程师

职位描述
1. 负责出门问问HTML5版本的开发
2. 负责内部web工具的开发

职位要求
1. 精通W3C标准的网站构建并有成功案例，精通HTML、CSS，熟悉HTML5和CSS3等新技术
2. 熟悉Javascript及主流JS库
3. 对Web界面的性能优化有丰富经验，对界面与交互体验有像素级的要求，对业界最新的前端技术和实现有浓厚的兴趣及深入的见解
4. 精通HTML5者和有框架开发经验者优先考虑
素质要求
1. 适合快速变化的工作环境，良好的团队合作能力
2. 逻辑分析能力强，学习能力佳，善于沟通

后端工程师

职位描述
1. 为基于互联网的服务编写服务端代码
2. 在服务器端提供快速稳定的服务,为用户的自然语言查询提供精准有效的答案

职位要求
1. 熟悉Java或者Ruby (Ruby：要求一年以上 Rails 开发经验) (Java：熟悉Java网络编程，至少写过10000行Java代码)
2. 熟悉 MySQL, Redis
3. 扎实的算法与数据结构基础
4. 有维护 Github 开源项目者,ACM竞赛或者各种编程创新大赛上有获奖经历者优先考虑

素质要求
- 极强的独立思考能力，做事专注细致

系统工程师

职位描述
1. 监控并保证整个系统的稳定运行, 用各种脚本掌控整个系统
2. 优化上线和部署流程，提供线上自动化支持，优化可能的薄弱环节

职位要求
1. 精通 Linux 系统及其内核
2. 掌握shell/python，及其在系统管理上的应用
3. 熟悉常用的服务器和硬件，并了解如何优化
4. 熟悉系统监控和优化的各类开源技术和工具
5. 熟悉mysql或者nosql的调优的优先
6. 在各种云计算平台管理过应用和账户优先

素质要求：极强的责任感

Android / iOS 工程师

职位描述
1. 负责出门问问Android/IOS版本的开发，测试和维护
2. 记录与统计App的用户数据与行为

职位要求
1. 了解MVC和常用设计模式，对代码质量和编程规范有较高要求
2. 有上线App者优先考虑

Android：
1. 能独立开发Android应用，有上线App尤佳
2. 熟悉 Java, Android SDK/NDK，熟练使用Linux/Unix，了解网络和数据库

iOS：
1. 熟练使用Objective-C或C/C++，独立开发过App，有上线App尤佳
2. 熟练使用Mac，了解网络和数据库，熟悉iOS内存管理机制和多线程编程

素质要求
团队合作能力、解决问题能力强

移动搜索，改变世界

15年前，Google在Stanford地下室诞生，打破了传统的信息获取方式，信息搜索革命的狂潮随之席卷全球。身处这个伟大的时代，我们在为之欢呼与奋斗的同时，也时常思考：究竟什么是未来的搜索？

如果，移动搜索是我们撬动这个世界的杠杆，那么那个支点，我们希望有你和我们一起撑起。

加入我们。用你的思考和行动，和我们一起改变世界。
