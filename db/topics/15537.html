工作地点: 浦东张江

招聘计划:
2-3名Ruby开发/技术主管

要求(Ruby开发/技术主管)

  1. 熟悉ruby，具备良好的编程基础，追求代码质量；
  2. 半年以上Ruby on rails开发经验；
  3. 半年以上HTML, Javascript经验
  4. 能熟练阅读英文技术文档；富有团队精神,责任感和沟通能力
  5. 拥有geek精神者优先考虑, 熟悉以下工具者项者优先考虑: Git, Redis, MongoDB, Postgres,Neo4j, Resque, Sidekiq
  6. 有以下经验者优先考虑: Machine Learning， NLP(natural language processing)，Speech Recognition，Big data visualization

注：1)该Ruby开发职位比较看重应聘者的学习能力，所以欢迎刚毕业不久的同学来尝试

薪资范围（根据能力以及工作年限)
工程师：
基本月薪:8k -12k， 全年13薪,年终奖另算，

技术主管：
基本月薪:12k -18k， 全年13薪，年终奖另算

特别优秀者待遇可以再议

其他福利：15天带薪年假+12天带薪病假，公司加5金1险（公司出你月薪的6%作为补充养老金），以及健康保险（自己或者家人看病发生的费用除医保支付外都可以另外报销) 每周多次健身活动（包场的羽毛球，篮球，游泳，瑜伽等），每年还有旅游。

当然我们的待遇在市场上不算好的，也没有富裕的妹子资源，但是大公司毕竟稳定各方面也很完善，对该职位感兴趣的同学请邮件请联系bao1018@gmail.com
