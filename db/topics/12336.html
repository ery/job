职位介绍

凯业必达(CareerBuilder)是北美最大的在线网络招聘运营商，也是全球流量最大 的30个网站之一。我们正在招聘Ruby on Rails 软件工程师，前段开发工程师。我们鼓励员工的自主创新精神，帮助员工不但在工作上有机会发挥自己的创造性，同时也为员工提供良好的学习与工作的环境。凯业必达被Glassdoor.com评为2012年芝加哥50大最佳雇主之一，这也是凯业必达连续第四年获此殊荣。在50家公司中，凯业必达排名第6位，福利除了正常的社保和公积金外，我们还提供健身福利补助，额外家庭人寿保险，领导培训机会和出国出差机会。

以下是我们正在用到的几种技术：

  1. Ruby on Rails, Heroku
  2. C#, VB.NET, ASP.NET
  3. SQL, Postgres
  4. Git, TFS
  5. Web Services, CSS 3.0, JQuery, JSON
  6. 任何技术只要能让我们为Business提出更好的解决方案.

Rails 软件工程师职位需求：

  • 熟悉 Ruby 语言和 Ruby on Rails 框架
  • 熟悉面向对象编程，逻辑能力强
  • 主动学习并且热爱新技术
  • 有应用编程语言分析和解决问题的能力
  • 善于沟通，具备良好的团队合作精神
  • 能够按照商业需求提出好的解决方案
  • 英语熟练，CET-4及以上

企业介绍：

凯业必达(CareerBuilder)是北美最大的在线网络招聘运营商，也是全球流量最大 的30个网站之一。目前已在美国、英国、法国、瑞士、意大利、德国、印度和 中国等18个国家设立公司，覆盖了全球55个国家和地区。凯业必达为全球的求职者提供海量的就业机会，也为包括世界500强企业、中小企业，以及政府部门提供了先进理念的招聘通道。通过世界上最先进的”快速搜索”引擎技术与专有的职位匹配技术，凯业必达帮助雇主与求职者克服所用术语差异，增加求职者了解相关工作以及雇主获得更合格候选人的机会。 凯业必达(CareerBuilder)拥有统一的全球化平台，世界上最先进的“快速搜索” 引擎技术及专有的匹配技术，可扩展的合作伙伴模式。在全球拥有超过2000家的合作伙伴。

2009年3月取得了中国政府批准并颁发的首张中外合资在线招聘牌照，正式进入中国市场。在中国已与上海热线形成战略合作伙伴关系，联合建立上海热线招聘频道。为了配合中国市场的开拓和长期发展，欢迎你加入我们的大家庭，和我们一起创造一个精彩的未来。

请将简历邮件发送至 liang.huang (a) careerbuilder.com
