[Cyberspace] Enumerable 小组招募 90 后 Web 开发爱好者

招聘 • alsotang • 于发布 • 最后由 rankjie 于回复 • 1709次阅读

大家好！我是 Enumerable 小组的联合发起人 @alsotang。

之所以想创建这么个团队，与这个帖子有一定的关系： 《技术大牛@曹政（caoz）曾经有一个神秘的QQ群》 http://www.zhihu.com/question/20807920 。

我相信，@caoz 当年在创建这个 QQ 群的时候，一定也还是一个默默无闻的小子，他只是凭着自己对于互联网的热情，找了一些同样对互联网有热情的小子一起，创建了这样一个 QQ 群。之后，这个 QQ 群里的石头不断相互碰撞打磨，最终一个个都变得美丽光滑。（磨石头的比喻引自《乔布斯：遗失的访谈》： http://www.huxiu.com/article/14067/1.html ）

很多朋友或许想象过像当年的@caoz 一样，找到一批志同道合的朋友，与他们一起在互联网世界中探索和玩乐，但却没有付诸实践。而这篇招募贴，正是对这一想法的实践。

我先来介绍一下自己，以及另外一位联合发起人 @hit9 。

@alsotang 的介绍在此： http://fxck.it/about

@hit9 的介绍在此： http://hit9.org/about.html

@hit9 在哈工大，我在川大，我们从未见过面。如果你加入了我们，那或许大家也没机会见面。

如果在用心看了我们的博客和 GitHub 后，你觉得自己与我们对于互联网的热情是一样的，那么你应该就是我们要寻找的人。：）

招募要求：

  1. 必须是 90 后。

  2. 高于同龄人的 Web 开发编码水平，并喜欢使用 Unix-like 操作系统。（通过 GitHub 来证明）

  3. 喜爱表达与分享。（通过博客来证明）

  4. 愿意奉献时间参与到开源项目的开发中。

工作职责：

  1. 积极发起并主导一个在 GitHub 上开源的有意思的项目。

  2. 积极参与小组其他成员在 GitHub 上开源的有意思的项目。

待遇：

  1. 薪水：0

  2. 户口：不帮解决

  3. 免费的饮料和水果：无

  4. 工作方式：SOHO

  5. Mac：不配备

  6. 地点：Cyberspace

届时，你会被加入到一个 Google Groups 的邮件列表中来，我们通过邮件、Dropbox 和 GitHub 来进行协作。这个邮件列表的上线人数是 10 人，因为我们认为一个小圈子里面的人会比较互相了解和熟悉，沟通成本低。由于圈子较小，大家也会更有责任感地参与到圈中人发起的开源项目活动中。

联系我们：

我的邮箱是 alsotang@gmail.com 。有意愿的朋友，请发送你的 GitHub 地址与博客地址过来，无需简历。：）

FAQ：

  1. 为什么团队叫 Enumerable？

Enumerable 是 Ruby 中的一个核心 module。如果一个自定义类实现了 :each 这个实例方法，那么就可以通过 include 这个 module，来获得诸如 [:map, :reduce, :select, :sort] 之类的许许多多迭代方法。
