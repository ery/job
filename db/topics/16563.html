关于淘淘搜

淘淘搜，成立于2010年4月，是全球领先的图像购物搜索引擎！

淘淘搜的投资方，包括全球最大的电子商务集团阿里巴巴，和国内最优秀的投资机构鼎晖、德同。

淘淘搜前身搜图购，是“赢在淘宝”2009年TOP应用大赛总冠军，该大赛是迄今为止，国内最大的开发者大赛，吸引了超过4000多个开发者团队或个人参赛，覆盖了电子商务产业链上下游的全部应用；淘淘搜也是创业邦2010年度中国创新成长100强！

淘淘搜经过三年的努力，现在已经是全球已知图像搜索领域应用规模最大的公司，累计使用用户过亿，每周活跃独立用户超过400万。收录在线商家数达到数百万，商品数已超过两亿，商品图片数近10亿！

高级 NodeJS 开发工程师

招聘职位：高级WEB开发工程师（Nodejs方向）

招聘人数：1人

薪资标准：10K~15K

岗位描述：

  1. 负责高流量、高并发的在线业务的日常需求；

  2. 负责建立基于Nodejs的web框架；

  3. 负责优化在线业务的自动化测试和发布流程；

  4. 负责开展开源项目以及社区活动，提升团队影响力和团队素质；

工作年限：1年及1年以上。

岗位要求：

  1. 熟悉Linux/Unix系统以及shell脚本；

  2. 具备一年以上Nodejs开发经验；

  3. 熟悉敏捷开发流程；深谙自动化测试的重要性；

  4. 具备Mysql或Mongodb使用经验，并能进行简单的数据库优化。

  5. 重视团队写作，愿意改进和完善开发流程，掌握常用工具（Git和CI系统等）。

  6. 具备开源社区开发经验者优先。

高级Javascript工程师/Javascript Ninja

招聘人数：1名

薪资标准：10K~15K

岗位描述：

  1. 负责淘同款业务日常需求；

  2. 负责建立淘同款主要产品需求的自动化测试方案。确保核心逻辑能够持续稳定上线；

  3. 负责在高访问量业务场景下，对前端代码进行性能优化；并能配合后端开发准备容灾预案；

  4. 负责产品原型和用户体验设计的评估工作，参与交互设计，分析影响产品用户体验因素。

工作年限：2年及2年以上。

岗位要求：

  1. 精通各种Web前端技术，包括HTML/CSS/Javascript等；

  2. 持续关注业界的新话题和新技术，研究过AngularJS、Ember、Backbone等框架中的一种或以上；

  3. 重视团队协作，愿意探寻和改进目前还不成熟的前端开发流程，了解各种常用工具（如Git、Grunt等）；

  4. 至少熟悉一门非Web前端脚本的语言（如Java/PHP/C++）或Node.js，并有实践经验；

  5. 个性乐观开朗，逻辑性强，善于和各种背景的人合作”。
