[上海] 急切需要 Ruby On Rails 的内行高手

招聘 • np_yao • 于发布 • 最后由 iBachue 于回复 • 928次阅读

我们急切需要Ruby On Rails的内行高手，来协助我们把既有产品推动到更新的台阶。您富有才能和创意，能独立思考和工作，是一位熟练的敏捷开发实践者。您想构建能被上百万人使用的软件平台，有对产品施加积极影响的强烈愿望。

关于我们：
NOOK Media 隶属Barnes&Noble。它是一个全球性的公司，宗旨是运用现代技术来改变人们通常的阅读和写作习惯。由于它拥有和出版商的广泛联系，具备独有的联络读者和学生的能力，并且设计和销售众口皆碑的NOOK 设备，NOOK Media 有很大的发展余地，将成为数字内容的领头者。

工作环境：
- 我们提供极具竞争力的薪资福利；
- 我们还可以资助您参加相关的技术会议和深造课程；
- 我们的办公地点位于上海徐汇区最繁华的商业地段，近地铁1号线；
- 作为一名全职员工，您将和我们在美国硅谷的其他技术人员保持紧密的协作

职位要求：
- 精通Ruby 和 Ruby On Rails 的开发；
- 熟悉Ruby 的数据序列化；
- 精通程序调试;
- 熟练掌握Capistrano ；
- 熟悉Linux CLI;
- 熟悉Resque;
- 熟悉现今流行的数据库和缓存技术, 如MySQL，Redis，memcache ;
- 熟悉运用RSpec/FactoryGirl编写单元测试程序;
- 善于编写明了，高效，可扩展的软件代码
- 熟悉项目版本控制工具，特别是Git;
- 熟悉事件驱动和异步等编程概念及实际应用；

关于您：
- 可靠, 有很强的上进心；
- 勤奋好学，热衷于解决疑难问题；
- 独立思考，有执行力，具备团队精神，与他人共进；
- 有较强的英语交流能力，包括阅读，口头，书面；

关于团队：
NOOK Press 是 NOOK Media 旗下一个正快速成长的重要的组成部分。成千上万作家的参与已成为公司收入的重要来源。我们正加倍努力，将给独立出版业带来革新。

联络方式：
有兴趣者，请踊跃报名。请将英文版求职信并简历通过电邮发送至 tjackiw@barnesandnoble.com. 电邮主题必须是英文：[Shanghai] Senior Ruby On Rails Engineer Application. 我们期待收到您的来信。
