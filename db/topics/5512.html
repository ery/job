[北京] 友盟诚聘中高级运维工程师以及 Ruby/Java/Scala 工程师

招聘 • wangyuxi • 于发布 • 最后由 woaigithub 于回复 • 1084次阅读

友盟 (http://www.umeng.com) 源于李开复博士创立的创新工场，已获得创新工场及国际知名风投的千万美金投资。移动互联网的大潮已经来临，友盟以移动应用实时分析服务平台切入，致力于为广大移动开发者提供全方位的服务，取得了不错的进展，之后我们将继续努力，除了做好分析平台，也会推出为移动 app 和开发者定制的云服务 mobile cloud service) 及移动广告平台。

我们是一个有活力和实力的年轻团队，CEO 和 CTO 都来自 Google, 其中 CTO 曾在 Google 美国总部核心部门工作多年；我们的核心团队成员多来自业界知名 IT 公司如 Baidu, Sohu 等。我们经常会内部组织分享技术讨论，互相学习，一起提高技术实力。

我们希望招一名中高级的运维工程师，跟我(wangyuxi)起共事，一起见证由上百台到上千台服务器的质变过程。

职责描述：
1. 熟悉并掌握网站系统架构的设计，进行必要的优化，保证网站的稳定
2. 对服务器运行状况，负载，应用系统运行状况等进行监控
3. 编写系统运维手册，完善相关的手册文档
4. 探索，研究新的运维技术，跟进社区新技术

任职资格：
1. 5 年以上使用某种发行版本 (Arch, Gentoo 尤佳) 做为主力 desktop 或者你是狂热的 Mac 粉，3 年以上一线运维经验
2. 熟悉常见的 Nginx，Tomcat，LVS，Keepalived，Heartbeat，HAProxy，Zabbix 等服务
3. 有很强的故障判断分析处理 (troubleshoting) 能力
4. 精通 Linux 网络，对常见协议有比较深刻的理解
5. 熟练掌握交换、路由和防火墙的配置
6. 熟悉内核调优，能解决常见的问题
7. 熟悉 puppet 等自动化工具，熟悉虚拟化 (xen, KVM)
8. 熟练掌握 1-2 门脚本语言，bash，ruby 尤佳
9. 有良好的英文阅读能力，善于沟通，乐于分享，喜欢挑战

加分项：
1. 有独立技术博客
2. 在社区报过 bug，关注跟进社区开源产品，发布过开源的作品
3. 狂热的科学上网控

除了上面这个之外，我们还需要下面的人才加盟。

* Ruby 工程师
职位要求：
1. 熟悉 Ruby，研究过 Rails 相关源码, 一年以上 Ruby/Rails 项目经验 (或者熟悉 Java, 基础好，对 Ruby 感兴趣的工程师)
2. 熟悉 Javascript
3. 计算机科学或相关专业本科或以上学历
4. 扎实的计算机专业基础知识
5. 热爱程序设计，对互联网行业充满激情，有独立解决问题的能力。

加分项:
1. 参与过开源项目
2. 有 Java, Scala 或 C++ 项目经验
3. 熟悉 MongoDB/MySQL/Redis/Cassandra/Hadoop/Hbase
4. 熟悉大规模数据处理、高并发或分布式系统相关知识，有后端服务优化相关经验

* Java/Scala 工程师
职位要求：
1. 熟悉 Java/Scala/Clojure
2. 有互联网后端项目开发经验
3. 计算机科学或相关专业本科或以上学历
4. 扎实的计算机专业基础知识
5. 热爱程序设计，对互联网行业充满激情，有独立解决问题的能力。

加分项:
1. 参与过开源项目
2. 熟悉 MongoDB/MySQL/Redis/Cassandra/Hbase/Hadoop/MapReduce
3. 熟悉大规模数据处理、高并发或分布式系统相关知识， 有后端服务优化相关经验

我们中午提供午饭，饭后酸奶，水果，下午小吃零食，海底捞牡丹园店就在我们楼下；桌上足球，Kinect，高级按摩椅等一应俱全；公司上下班不打卡，定期组织看电影外出等活动；提供五险一金。
